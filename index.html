Cancer Genomics Analyzer

A web application for analyzing VCF genomic data, estimating cancer-associated risk, and presenting key mutation insights.

View on GitHub Read Documentation

What It Does

  • Parses VCF variant data
  • Highlights high-impact genes
  • Provides risk-oriented summaries

Tech Stack

  • Python + Flask backend
  • HTML/CSS frontend templates
  • JSON APIs for analysis results

Important Note

  • This GitHub Pages site is static
  • Live analysis requires running the Flask app locally or on a server
For full functionality, run python app.py and open http://127.0.0.1:5000. This page is a project landing page for GitHub hosting.
